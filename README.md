# QuizMaster

QuizMaster is a responsive frontend quiz application built using HTML, CSS, Bootstrap, and JavaScript.

## Features
- Category-based quizzes
- Score calculation and result view
- View past and future quiz scores
- Responsive UI

## Login Note
Login is implemented only as a UI-level access control to display the score history page.  
This project does not include backend authentication or database integration.

## Tech Stack
- HTML
- CSS
- Bootstrap
- JavaScript

## How to Run
Open `index.html` in a browser.
